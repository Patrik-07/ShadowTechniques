# OpenGL Game Engine

Supports:
   - loading 3D models
   - play animations (Skeletal aniamtion)
   - basic lightning (Phong)
   - shadow mapping
   - ambient occlusion (SSAO, HBAO)
   - forward and deferred rendering
   - multiple render window targets (FramwBuffers)
   - Custom uniform setting system
   
Used technologies:
   - C++
   - OpenGL
   - SDL2
   - Assimp
   - Imgui
   - GLM
